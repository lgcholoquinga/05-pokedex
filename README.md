# run in mode development
1.- Clone repo
2.- run 
```
npm install
```
3.- Do you have Nest CLI installed
```
npm install -g @nestjs/cli
```
4.- Rub database
```
docker-compose up -d
```

## Stack
* MonfoDB
* NestJs
* Docker
